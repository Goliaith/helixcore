How the System Works

1. Local-First Inference
All reasoning runs on local Ollama models (configured in E:\AI\Grok\config.toml). No cloud calls unless you explicitly request "full power".
2. Governed by HelixCore
Major actions use begin_governed_work, phase handoffs, persist_decision, and local semantic memory for structure and observability.
3. Hybrid Tool Use
Pure local tools (LocalCodeIntel, semantic memory) are preferred. MCPs (chroma, cognee, context7) are used as supplements via search_tool then use_tool when useful.
